# AEB (Agência Espacial Brasileira) - Sensoriamento Remoto


Ao final do curso, espera-se que os participantes compreendam de maneira estruturada os conceitos essenciais dessa tecnologia, desenvolvendo uma base sólida sobre os diferentes tipos de sensores, as principais plataformas de satélites e os elementos que compõem uma imagem geoespacial. 
Também se espera que adquiram capacidade de interpretar imagens de satélite com precisão, aplicando práticas de análise para identificar padrões e fenômenos geográficos. Esse processo permitirá reconhecer alterações na cobertura vegetal, transformações decorrentes da urbanização e mudanças em corpos d’água, promovendo o aprimoramento de habilidades técnicas e investigativas fundamentais para o uso adequado de informações espaciais. 


## Conteúdo:

- História do Sensoriamento Remoto;
- O que é sensoriamento remoto?;
- Comprimento de onda - reflexo de energia;
- Característica espacial;
- Característica temporal;
- Característica espectral, radiométrica, sensor mux e pan;
- Interpretação visual de imagens;
- Aplicações - urbano e patrimônio cultural;
- Aplicações - desastres;
- Aplicações - recursos hídricos;
- Aplicações - vegetação;
- Exemplo de uso na plataforma Copernicus.
